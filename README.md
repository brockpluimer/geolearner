# 🌍 geolearner

An interactive world-geography quiz. Learn countries, capitals, and flags one
region at a time — the app tracks which countries you miss and quizzes you on
them more often.

Built with **Vite + React**, an SVG world map rendered with **d3-geo** from
**Natural Earth** (1:50m admin-0) TopoJSON, and flag images from
[flagcdn.com](https://flagcdn.com).

## Modes

| Mode | Prompt | Answer |
| --- | --- | --- |
| **Map → Name** | A country lights up on the map | pick / type its name |
| **Flag → Country** | A national flag | pick / type the country |
| **Country → Capital** | A country name | pick / type its capital |
| **Capital → Country** | A capital city | pick / type the country |
| **Find on Map** | A country name | click the right shape on the map |

- **Difficulty toggle** — multiple choice (3 distractors drawn from the same
  region) or typed answers with fuzzy matching for typos and aliases
  (`USA` → United States, `UK` → United Kingdom, …).
- **Region filter** — quiz a single continent or the whole world.
- **Adaptive selection** — miss counts are saved to `localStorage` and weight
  question selection toward your trouble spots.
- **Scoring** — live score, current streak (🔥 at 3+), best streak, and an
  end-of-round summary with a review of your misses.

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## Data

- `src/data/world-50m.json` — Natural Earth 1:50m countries as TopoJSON
  (via the [`world-atlas`](https://github.com/topojson/world-atlas) package).
- `src/data/countries.json` — country metadata keyed by ISO alpha-3
  (name, capital, region, ISO numeric map id, flag code), generated by
  `node scripts/build-countries.mjs` from the
  [`world-countries`](https://github.com/mledoze/countries) dataset.

`scripts/verify.mjs` is a headless end-to-end smoke test (needs a local dev
server and `npm i --no-save puppeteer-core`).
